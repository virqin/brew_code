Ajuda

Utilize a seta para cima e a seta para baixo para se deslocar na vertical e alterar a escolha de ligação. Prima o botão de selecção para seguir uma ligação ou para preencher um campo de formulário.

Prima a seta para a esquerda ou a seta para a direita para aceder à barra de ferramentas, que tem os seguintes botões:

  •   Retroceder leva-o à página anterior.
  •   Actualizar carrega novamente a página actual.
  •   Ir para permite introduzir qualquer URL (o BREW Browser recorda-se dos últimos que foram introduzidos).
  •   Informações pode prestar mais indicações acerca da página actual.
  •   Marcadores permite gerir os marcadores.
  •   Ajuda devolve-o a este esclarecimento.
  •   Opções permite configurar o BREW browser.

Divirta-se!
